# CtrlP UI+ Backend

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Copy logo files to the assets folder:
- Place your `logo.png` and `logo-light.png` files in the `src/assets` directory

3. Start the application:
```bash
npm start
```

## Development
Run in development mode:
```bash
npm run dev
```

## Build for production
```bash
npm run build
```
